# Code of Conduct

This account's public repositories are intended to be respectful, practical,
and useful to other engineers.

## Our standards

Examples of behavior that helps keep the community constructive:

- being respectful and professional
- assuming good faith
- giving actionable technical feedback
- reporting bugs with clear reproduction details when possible
- helping keep discussions focused on the repository topic

Examples of unacceptable behavior:

- harassment, abuse, or personal attacks
- discriminatory or hateful language
- spam, trolling, or intentionally disruptive behavior
- publishing private information without permission
- derailing technical discussions in bad faith

## Scope

This Code of Conduct applies to issues, pull requests, discussions, review
comments, and other public interactions in repositories under the
`happysnaker` account.

## Reporting

If you experience or observe behavior that violates this Code of Conduct,
please report it via:

- `happysnaker@foxmail.com`

## Enforcement

Unacceptable behavior may result in comment removal, issue/PR closure, or
blocking future interaction as appropriate.
